Tags Package

The SparForte built-in tags package provides subprograms to manipulate value meta tags.

Introduced: SparForte 3.0

GCC Ada Equivalent: N/A

  b := contains_policy( x, t )    b := contains_unit( x, t )      s := get_policies_image( x )
  s := get_unit_image( x )        b := has_policies( x )          b := has_unit( x )
 
Help Command: Contents of the tags package

b := tags.contains_policy( x, t )

 

Return true if the given tag is in the policies of x.

Example

 credit_card := "1234567890" tagged pci;
b := tags.contains_policy( credit_card, pci ); -- returns true

Parameters
Param Mode Type Default Description
x in any scalar type required the item with the value to test
t in meta policy required the policy to check
b return value boolean required true if the value has the policy

Exceptions

An error occurs if the tag is for a unit of measure

b := tags.contains_unit( x, t )

 

Return true if the given tag is the unit of measure for x.

Example

 temperature := 20.0 tagged celsius;
b := tags.contains_unit( temperature, kilometers ); -- returns false

Parameters
Param Mode Type Default Description
x in any scalar type required the item with the value to test
t in meta unit of measure required the unit of measure to check
b return value boolean required true if the value has the unit of measure

Exceptions

An error occurs if the tag is for a policy

s := tags.get_policies_image( x )

 

Return the policy tags of x as a human-readable string. If there are no policy tags, an empty string is returned.

Example

 x := "1234567890" tagged pci;
s := tags.get_policies_image( credit_card ); -- returns "pci"

Parameters
Param Mode Type Default Description
x in any scalar type required the item with the value
e return value string required the string of the policies

s := tags.get_unit_image( x )

 

Return the unit of measure tag of x as a human-readable string. If there is no unit of measure tag, an empty string is returned.

Example

 distance := 100.0 tagged miles;
s := tags.get_unit_image( distance ); -- returns "miles"
