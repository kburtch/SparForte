Command Reference

These are other miscellaneous AdaScript commands. These are not built-in commands but instead are derived from the Ada 95 programming language.

command [obsolete/prior to Bush 1.0]

Syntax: command cmd
Description: run a Linux command (instead of a built-in command).

delay

Syntax: delay secs
Description: wait (sleep) for a specific time.

null

Syntax: null
Description: do nothing. This command can be used as a placeholder in places where a command is required.
