Vectors Package

A collection of common routines for working with dynamic arrays (indexed lists).

Introduced: SparForte 2.4.2

GCC Ada Equivalent: Ada.Containers.Vectors

Help Command: Contents of the vectors package

This package includes new generic types:

  • vectors.cursor( key type, element type ) - a cursor for iterating through a vector
  • vectors.vector( key type, element type ) - a dynamic array

Ada: Because AdaScript does not have subtypes with ranges, you cannot change the first index or last index. For example, any type based on positive will start at index 1.
 
Example:

hashed_maps.append( m, k, e )

 

Append string element e to the value under key k in the hash map. If the key does not exist, raise an exception.

Example

hashed_maps.insert( fruit_unit, "apple", bushel" );
hashed_maps.append( fruit_unit, "apple", "s" ); -- changed to "bushels"

Parameters
Param Mode Type Default Description
m in out hashed_maps.map required the map
k in map key type required the key
e in map element type required the string to append

Exceptions

An exception is raised if the key is the wrong type
An exception is raised if the key does not exist
An exception is raised if out of memory
An exception is raised if the element type is not a string

Restrictions

Not allowed with pragma ada_95

See Also

hashed_maps.prepend

Compare With

Implementation Note

This is in accordance with memcached.

hashed_maps.assign( t, s )

 

Assign source map s to target map t, overwriting the contents of t. s is unchanged.

Example

hashed_maps.assign( old_fruit, updated_fruit );

Parameters
Param Mode Type Default Description
t in out hashed_maps.map required the map to be assigned to
s in out hashed_maps.map required the map to copy from

Exceptions

An exception is raised if the key or element has the wrong type
An exception is raised if out of memory

Compare With

Ada: Ada.Containers.Hashed_Maps.Assign
Java: HashMap.putAll()

Implementation Note

Assign fails on CentOS 7 due to a Ada bug.

hashed_maps.delete( m, k | c )

 

Remove a key and the associated element from the hash map. If a cursor is given instead of a key, delete the item at the cursor position.

Example

hashed_maps.delete( fruit, "bad apples" );

Parameters
Param Mode Type Default Description
m in out hashed_maps.map required (if no cursor) the map
k in map key type required the key to delete
c in map element type required (if no map) the cursor position to delete

Exceptions

An exception is raised if nothing is at the cursor position.
An exception is raised if the key is not found in the map.

Restrictions

Not allowed with pragma ada_95

See Also

hashed_maps.reset
hashed_maps.set

Compare With

Ada: Ada.Containers.Hashed_Maps.Delete

Implementation Note

Ada uses "Delete" but delete is a reserved word in SparForte. This may be renamed in the future.

hashed_maps.insert( m, k, e ) | ( m, k, c, b ) | ( m, k, e, c, b )

 

Insert the value v under key k in the hash map. If b exists, b is true if the insert was successful. If value e exists, set the value to e. If c exists, move the cursor to the insert position. If the key exists, raise an exception.

Example

hashed_maps.insert( fruit, "apples", 23 );

Parameters
Param Mode Type Default Description
m in out hashed_maps.map required the map
k in string required the key
e in table value type no value the value
c out hashed_maps.cursor optional the cursor
b out boolean optional true on success

Exceptions

An exception is raised if the value is the wrong type
An exception is raised if out of memory
An exception is raised if the key exists

See Also

hashed_maps.set
hashed_maps.add

Compare With

Ada: Ada.Containers.Hashed_Maps.Insert

Implementation Note

Ada uses "Insert" but insert is a reserved word in SparForte. This may be renamed in the future.
