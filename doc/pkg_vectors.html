Vectors Package

A collection of common routines for working with dynamic arrays (indexed lists). This is based on the work of Grady Booch.

Introduced: SparForte 2.5

GCC Ada Equivalent: Ada.Containers.Vectors

Help Command: Contents of the vectors package

This package includes new generic types:

  • vectors.cursor( index type, element type ) - a cursor for iterating through a vector
  • vectors.vector( index type, element type ) - a dynamic array

This package includes a constant:

  • vectors.no_index - a long integer value indicating no position

An empty vector has a value of two negative values, the last lower than the first. These values are long_integers and may be out of the range of for your index type.

Ada: Because AdaScript does not have subtypes with ranges, you cannot change the first index or last index. For example, any type based on positive will start at index 1.

SparForte does not have an Extended_Index type.
 
Example: To be created

vectors.append( v, i, e )

 

Append string element e to the value under index i in the vector. If the index does not exist, raise an exception.

Example

vectors.insert( fruit_unit, "apple", bushel" );
vectors.append( fruit_unit, "apple", "s" ); -- changed to "bushels"

Parameters
Param Mode Type Default Description
v in out vectors.vector required the vector
i in vector index type required the index position
e in vector element type required the string to concatenate

Exceptions

An exception is raised if the index is the wrong type
An exception is raised if the index does not exist
An exception is raised if out of memory
An exception is raised if the element type is not a string

Restrictions

Not allowed with pragma ada_95

e in vector element type required the element to queue
n in containers.count_type 1 the number of copies

Exceptions

delete( v, c ) | ( v, i [, n] )

 

Remove an element from the vector at the index position, effectively sliding the elements toward the start of the vector by one position. If a cursor is given instead of an index, delete the item at the cursor position. If a count of n is given, delete n elements instead of one.

Example

-- if char_vector contains "a", "b", "c"
vectors.delete( char_vector, 2 ); -- vector is "a", "c"

Parameters
Param Mode Type Default Description
v in out vectors.vector required the vector
i in vector index type required (if no cursor) the position to delete
c in vectors.cursor required (if no index) the cursor position to delete
n in containers.count_type 1 the number of elements to delete

Exceptions

An exception is raised if the number of elements is not an integer-type value

See Also

vectors.find_index( v, e, i1, i2 )

 

Return the index in the vector for element e. Start at the position of index i1. If the element does not exist, the index i2 will have No_Index. A long_integer is returned and the user can typecast it or use enum.val to convert it to the index type (unless it is No_Index).

Example

vectors.find_index( device_vector, "firewall", 1, position_index );

Parameters
Param Mode Type Default Description
v in out vectors.vector required the vector
e in element type required the element to find
i1 in out vector index type required the starting position
i2 out long_integer required the position of the element or no_index

Exceptions

An error occurs if the element is the wrong type
An exception is raised if the starting position value is bad

See Also
