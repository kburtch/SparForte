Welcome to SparForte

This is the official documentation for SparForte (that is, this is the SparForte manual). It has been written by the SparForte developers and volunteers. This manual has been organized into several parts or sections including tutorials, reference material, the design rationale and introduction information like frequently asked questions.

Within each section, topics will be listed in the side menu.

Start by selecting a section name from the menu. Below is a summary of the different sections.

Table of Contents

Step-by-step examples for new users to learn about SparForte. You start by learning how to use the command prompt.

This section contains detailed technical explanations of the core features and how they are to be used (excluding the built-in packages).
