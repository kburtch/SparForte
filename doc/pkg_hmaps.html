Hashed Maps Package

A collection of common routines for working with key-value tables (such as arrays in PHP, PERL hashes or dictionaries in Python).

Introduced: SparForte 2.4.2

GCC Ada Equivalent: Ada.Containers.Hashed_Maps

Help Command: Contents of the hashed_maps package

A hash table is a lookup table containing key-element (that is, key-value) pairs. Keys are unique in the table. The ken and element types must be scalar

This package includes a new generic type:

  • hashed_maps.cursor( key type, element type )
  • hashed_maps.map( key type, element type )

hashed_maps.add( m, k, e )

 

Put element e under key k in the hash map. If the key already exists, do nothing.

Example

hashed_maps.add( fruit, "apples", 15 );

Parameters
Param Mode Type Default Description
m in out hashed_maps.map required the map
k in map key type required the key
e in map element type required the value

Exceptions

An exception is raised if the key is the wrong type
An exception is raised if the element is the wrong type
An exception is raised if out of memory

Restrictions

Not allowed with pragma ada_95

See Also

hashed_maps.set
hashed_maps.replace

hashed_maps.append( t, k, e )

 

Append string element e to the value under key k in the hash table. If the key does not exist, do nothing.

Example

Parameters
Param Mode Type Default Description
m in out hashed_maps.map required the table
k in string map key type the key
e in map value type required the string to append

Exceptions

An exception is raised if out of memory
An exception is raised if the element type is not a string

hashed_maps.new_table( t, y )

 

(No longer available) Initialize a new hash table, assigning the type for the values in the table. The value type must be a scalar type, and it may be a universal type.

Example

hashed_maps.new_table( fruit, natural );

Parameters
Param Mode Type Default Description
t out hashed_maps.map required the table to initialize
y in the value type required the type of values

Removed
