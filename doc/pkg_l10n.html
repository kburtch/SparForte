L10N Package

The SparForte built-in l10n package provides routines for different languages and countries. This is a binding to the GNU locale functions.

Introduced: SparForte 2.5

GCC Ada Equivalent: none

  s := abday( p )        s := abmon( p )        s := amstr
  s := codeset           s := currency_symbol   s := day( p )
  s := decimal_point     s := d_fmt             s := d_t_fmt
  s := era               s := era_d_fmt         s := era_d_t_fmt

Help Command: Contents of the l10n package

The type locale_catagory is an enumerated type of the following:

  • l10n.lc_all - all locale categories
  • l10n.lc_collate - for string comparisons
  • l10n.lc_messages - for system messages
  • l10n.lc_monetary - for monetary values
  • l10n.lc_ctype - for character classes
  • l10n.lc_time - for date and time

The GNU extension to locale catagories are not included for portability reasons.

You may need to install additional languages on your operating system to use their locales. For example, with Ubuntu Linux, with Preferences /Languages.

On Linux, the locale command provides the same information.

=> ? l10n.getlocale( l10n.lc_all )
LC_CTYPE=en_CA.UTF-8;LC_NUMERIC=C;LC_TIME=C;LC_COLLATE=C;LC_MONETARY=C;LC_MESSAG
ES=C;LC_PAPER=C;LC_NAME=C;LC_ADDRESS=C;LC_TELEPHONE=C;LC_MEASUREMENT=C;LC_IDENTI
FICATION=C
=> ? l10n.currency_symbol

=> l10n.setlocale( l10n.lc_monetary, "en_US.UTF-8" )
=> ? l10n.getlocale( l10n.lc_monetary )
en_US.UTF-8
=> ? l10n.currency_symbol
$
=> ? l10n.int_curr_symbol
USD
=> ? l10n.day(1)
Sunday
=> l10n.setlocale( l10n.lc_time, "fr_CA.UTF-8" )
=> ? l10n.getlocale( l10n.lc_time)
fr_CA.UTF-8
=> ? l10n.day(1)
dimanche
 
Example: Using localization

s := abday( p )

 

Return the abbreviated day of the week for day number p. The first day is 1.

Example

 s := l10n.abday( 2 ); -- returns "Mon" (or similar)

Parameters
Param Mode Type Default Description
p in positive (1..7) required the day of week number
s return value string required the abbreviated day of the week

Exceptions

The day number is out-of-range
